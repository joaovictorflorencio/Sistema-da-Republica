# Sistema de República

Sistema desenvolvido para auxiliar na gestão de moradias compartilhadas, com foco no controle de despesas, divisão de tarefas e organização entre moradores.

---

## Objetivo

Fornecer uma solução que permita:
- Gerenciamento de despesas e pagamentos  
- Divisão automática de contas  
- Organização de tarefas domésticas  
- Controle de usuários e acesso  

---

## Público-Alvo

Estudantes e pessoas que vivem em repúblicas ou moradias compartilhadas.
